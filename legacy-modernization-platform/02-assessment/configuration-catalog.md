## 6. Migration Plan

### Configuration Migration

| Source | Target | Strategy | Effort |
|--------|--------|---------|:------:|
| web.config appSettings | `appsettings.json` + IOptions | Direct migration | 🟢 |
| web.config connectionStrings | Azure Key Vault / User Secrets | Externalize | 🟡 |
| Database config table | Azure App Configuration | API migration | 🟡 |
| Registry keys | Environment variables / Config | Externalize | 🟡 |
| Hardcoded constants | Configuration + IOptions | Refactor | 🔴 |

### Secret Migration

| Current Storage | Target | Migration Steps |
|----------------|--------|----------------|
| Config file | Azure Key Vault | 1. Create vault 2. Add secrets 3. Update code to use SDK 4. Remove from config |
| Source code | Azure Key Vault | 1. Identify all instances 2. Extract to vault 3. Use managed identity |
| Database | Azure Key Vault | 1. Map all secrets 2. Migrate 3. Update access pattern |
